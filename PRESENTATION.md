## 🧠 Алгоритм фильтрации сигналов

Сигналы отбираются по строгим условиям одновременно по 5 индикаторам:

**BUY сигнал** срабатывает когда:
- MACD < 0 (отрицательный тренд ещё не сломлен)
- MACD Histogram > 0 (гистограмма разворачивается вверх)
- RSI от 50 до 60 (зона умеренной силы, не перекуплен)
- ATR / Price × 100 > 3.5% (высокая волатильность — есть куда идти)
- EMA(20) < EMA(50) (краткосрочная MA ниже долгосрочной — поиск разворота)

**SELL сигнал** срабатывает когда:
- MACD < 0
- MACD Histogram < 0 (гистограмма продолжает падать)
- RSI от 35 до 40 (зона слабости без перепроданности)
- ATR / Price × 100 от 0.3% до 0.6% (контролируемая волатильность)
- EMA(20) > EMA(50) (краткосрочная MA выше — поиск шорта)

TP и SL рассчитываются автоматически на основе ATR.

## 🚀 Возможности платформы

### 1. Торговый терминал
- Полноценный интерфейс с TradingView графиком
- Живой стакан ордеров и лента сделок через WebSocket
- Открытие BUY/SELL позиций напрямую на Binance Futures
- Market и Limit ордера с настройкой плеча
- Управление открытыми позициями и ордерами в реальном времени
- Поддержка Cross и Isolated маржи
- Плечо от 1× до 125×

### 2. Автосигналы EMA + RSI + MACD
- Алгоритмические сигналы BUY/SELL на основе трёх индикаторов
- Автоматический расчёт Take Profit и Stop Loss через ATR
- Балльная система силы сигнала: STRONG / MEDIUM
- Автопроверка исполнения: статусы DONE / MISSED / PROCESSING
- Telegram-уведомления мгновенно при появлении сигнала
- История более 300 000+ сигналов в базе данных

### 3. Конструктор стратегий (новая фича)
- Создание собственных торговых стратегий с нуля
- 25+ технических индикаторов: RSI, EMA, SMA, WMA, MACD, ATR, Bollinger Bands, Stochastic, CCI, Williams %R, ADX, SuperTrend, Parabolic SAR, Ichimoku, VWAP, OBV, CMF, MFI, ROC, Keltner Channel, Donchian Channel, StdDev и другие
- BUY и SELL условия в одной стратегии — одновременно
- Логика AND / OR для комбинирования условий
- Запуск стратегии по конкретной паре или по всем 500+ USDT Futures парам сразу
- **Бэктест** на до 500 исторических свечах с месячной статистикой
- Режимы: Telegram-уведомление / Авто-трейд / Ручной
- Artisan-команда для автозапуска: `php artisan strategies:run`

### 4. Аналитика и результаты
- Ежедневная статистика: DONE / MISSED сигналы
- P&L расчёт с учётом TP и SL
- Win Rate по дням, неделям, месяцам
- Экспорт данных в Excel
- Сравнение периодов

### 5. Управление API профилями
- Несколько Binance API профилей в одном аккаунте
- Раздельные среды: PROD и TEST
- Реалтайм данные: баланс, открытые позиции, ордера, история сделок, PnL
- Управление позициями прямо из профиля (закрытие по рынку)
- WebSocket подключение к Binance User Data Stream

### 6. Крипто-новости
- Агрегация актуальных новостей криптовалютного рынка
- Фильтрация по монетам (BTC, ETH, SOL и другие)
- Поиск по ключевым словам
- Связанные статьи
